# Practice

This project was built almost entirely by [Claude Code](https://claude.ai/code).

## What is this?

A web app for practicing pronunciation in any language. It uses ElevenLabs text-to-speech to read sentences aloud, then lets you record yourself repeating them and compare.

## How to use

1. Enter your [ElevenLabs API key](https://elevenlabs.io/)
2. Pick a voice, speed, and language (or leave on auto-detect)
3. Paste or type some text and hit **Start**
4. The app let's use click on any sentence and:
   - Play the original via TTS
   - Record your own audio
   - Play your recording back
   - Play the original again so you can compare
5. Use arrow keys to navigate between sentences
6. Use player UI at the bottom to listen to the whole text at once
7. Hide the text itself, when you want to focus only on audio
